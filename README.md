# 🏦 Sistema Bancário em Java

Projeto desenvolvido com o objetivo de simular um sistema bancário simples, aplicando conceitos fundamentais de Programação Orientada a Objetos (POO) utilizando Java.

---

## 📌 📖 Sobre o projeto

O sistema permite a criação de contas bancárias e a realização de operações básicas como depósito, saque, transferência e consulta de extrato.

A aplicação é executada via console, com um menu interativo que permite ao usuário escolher as operações desejadas.

---

## ⚙️ 🚀 Funcionalidades

✔️ Criar contas bancárias  
✔️ Depósito de valores  
✔️ Saque com validação de saldo  
✔️ Transferência entre contas  
✔️ Exibição de extrato com histórico de transações  
✔️ Menu interativo no console  

---

## 🧠 💡 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Interfaces
- Enum (`TipoTransacao`)
- Listas com `ArrayList`
- Estruturas de controle (`switch`, `while`)
- Registro de transações
